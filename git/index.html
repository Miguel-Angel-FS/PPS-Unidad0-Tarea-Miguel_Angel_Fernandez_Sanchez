Documentación del proceso de creación del repositorio

Creación del repositorio en GitHub

El primer paso consiste en crear un nuevo repositorio dentro de GitHub.
Para ello vamos a la sección Repositories y pulsamos el botón New.

A continuación seguimos estos pasos:

  1. Introducir el nombre del repositorio.
  2. Activar la opción para generar automáticamente el archivo README.md.
  3. Crear el repositorio.

Estructura del proyecto

Una vez creado el repositorio, procedemos a configurar el sistema de archivos.
En este caso, la estructura del proyecto queda organizada de la siguiente forma:

PPS-Unidad0-Tarea-Tu_nombre/  
├── .github/workflows/  
│   ├── CreacionDocumentacion.yml 
├── docs/  
│   └── Miguel_Angel_Fernandez_Sanchez_Info_Contenedor.txt    
│   └── git.md  
│   └── gitActions.md   
│   └── gitPages.md  
│   └── docker.md  
│   └── conclusiones.md
│   └── index.md
├── mkdocs.yml
├── README.md

Dentro de la carpeta docs/ se encuentran todos los archivos de documentación del proyecto.
