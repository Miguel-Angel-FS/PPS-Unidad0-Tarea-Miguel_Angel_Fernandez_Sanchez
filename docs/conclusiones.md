# Conclusiones

Durante la realización de esta práctica he aprendido a utilizar diversas herramientas relacionadas con el control de versiones y la automatización de documentación. A continuación, destaco los principales conocimientos adquiridos:

- He creado y configurado un repositorio en GitHub siguiendo buenas prácticas de organización.
- He aprendido a estructurar adecuadamente la documentación utilizando archivos Markdown.
- He configurado GitHub Actions para automatizar la generación de documentación mediante MkDocs.
- He comprendido cómo funciona GitHub Pages y cómo desplegar una página web estática desde una rama específica del repositorio.
- He trabajado con workflows, detectando errores y comprobando logs para asegurar que todo se ejecute correctamente.
- He entendido la importancia de la automatización en los procesos de desarrollo, ya que facilita el mantenimiento y despliegue del proyecto.
